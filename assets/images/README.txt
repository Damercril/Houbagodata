Pour que l'application fonctionne correctement, veuillez :

1. Placer votre logo dans ce dossier avec le nom exact "logo.png"
2. Le logo devrait être au format PNG avec un fond transparent de préférence
3. La taille recommandée est de 240x240 pixels ou plus grand avec le même ratio

Note : Le logo est affiché avec une hauteur de 120 pixels dans l'application, donc assurez-vous qu'il soit lisible à cette taille.
